# Interactive 3D Data Demo

## Description
A prototype for visualizing real-world objects in 3D using Unreal Engine and Node.js.

## Use Case
The user sees objects on a 3D scene, can select them, and view their status.

## Data Structure (example object)
{
"id": 1,
"name": "Station #1",
"position": { "x": 120, "y": 75, "z": 0 },
"status": "active"
}
